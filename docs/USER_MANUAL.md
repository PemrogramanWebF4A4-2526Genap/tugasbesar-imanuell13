# User Manual Digital Zone

## Admin

1. Login menggunakan akun admin.
2. Masuk ke Dashboard Admin.
3. Kelola user serta verifikasi akun seller.
4. Kelola kategori produk dan data produk.
5. Kelola seluruh pesanan dan verifikasi pembayaran.
6. Mengubah pengaturan sistem pada menu System Settings.
7. Melihat statistik dashboard seperti jumlah user, produk, pesanan, omzet, produk terlaris, dan kategori terlaris.

## Seller

1. Register sebagai Penjual.
2. Tunggu akun diverifikasi oleh admin.
3. Login ke Dashboard Seller.
4. Tambahkan produk beserta gambar, kategori, harga, stok, dan masa garansi.
5. Edit atau hapus produk.
6. Kelola pesanan yang masuk dan ubah status pengiriman.
7. Lihat dashboard penjualan yang menampilkan total produk, total pesanan, produk terjual, dan omzet.

## Buyer

1. Register atau login sebagai buyer.
2. Cari produk menggunakan fitur pencarian atau filter kategori.
3. Tambahkan produk ke keranjang menggunakan fitur Cart (AJAX).
4. Kelola isi keranjang (update jumlah atau hapus produk).
5. Checkout dengan alamat pengiriman, metode pembayaran, dan jenis pengiriman.
6. Sistem otomatis menghitung ongkir dan membuat invoice.
7. Upload bukti pembayaran.
8. Pantau status pesanan melalui menu Pesanan Saya.
9. Berikan rating dan review setelah pesanan selesai.

## System Automation

- Invoice dibuat otomatis saat checkout.
- Ongkir dihitung otomatis berdasarkan jenis pengiriman.
- Stok produk berkurang otomatis setelah checkout berhasil.
- Status pesanan diperbarui setelah pembayaran diverifikasi.
- Notifikasi sistem dan email log dibuat otomatis pada proses checkout, pembayaran, dan perubahan status pesanan.
